# AI Hook Lab 设计文档

**日期：** 2026-05-23
**状态：** 已确认
**技术路线：** 纯前端单文件（HTML + CSS + JS），模板引擎生成，无需 API

---

## 1. 产品概述

AI Hook Lab 是一个面向内容创作者的爆款开头生成工具。用户输入主题，选择平台和内容类型，一键生成 10 个不同风格的 Hook（爆款开头文案）。每个 Hook 附带风格标签、点击率评分和推荐理由。

**目标用户：** 小红书/抖音/B站的内容创作者、自媒体运营者
**使用场景：** 不知道视频/图文怎么开头时，快速获取灵感

---

## 2. 技术架构

```
┌─────────────────────────────────────────┐
│           index.html (单文件)             │
├─────────────────────────────────────────┤
│  HTML: 页面结构                           │
│  CSS:  暖白极简风格，响应式布局             │
│  JS:   模板引擎 / 收藏管理 / 历史记录       │
├─────────────────────────────────────────┤
│  数据存储: localStorage                   │
│  - favorites:  收藏的Hook列表              │
│  - history:    最近20条生成记录            │
└─────────────────────────────────────────┘
```

- **零依赖：** 不引入任何外部库或框架
- **零网络请求：** 所有数据本地生成和存储
- **零构建：** 浏览器直接打开即可使用

## 3. 页面结构（从上到下）

| 区域 | 说明 |
|------|------|
| **Header** | Logo "⚡ AI Hook Lab" + 一句话介绍 |
| **输入卡片** | 主题输入框 + 平台选择（3个药丸按钮）+ 内容类型选择（5个药丸按钮）+ 生成按钮 |

## 4. 核心功能

### 4.1 输入区
- **主题输入框：** placeholder 轮播示例主题
- **平台选择：** 小红书 / 抖音 / B站（单选，药丸按钮样式）
- **内容类型选择：** 视频 / 图文 / 产品广告 / 教程 / 观点帖（单选）
- **生成按钮：** 全宽黑色圆角按钮，hover 有微动效

### 4.2 Hook 生成引擎（模板引擎）
内置 8 种风格 × 每种风格 5-8 个模板句式 × 3 平台变体 × 5 内容类型适配词 = 丰富组合空间。

**8 种 Hook 风格：**

| 风格 | 标签颜色 | 说明 |
|------|---------|------|
| 🔥 争议型 | 橙 #f97316 | 制造认知反差，引发讨论 |
| 📊 数据型 | 蓝 #3b82f6 | 具体数字增强可信度 |
| 😱 悬念型 | 紫 #8b5cf6 | 制造好奇缺口，促使用户看完 |
| 💡 干货型 | 绿 #22c55e | 承诺实用价值，直接解决问题 |
| 😂 共鸣型 | 粉 #ec4899 | 戳中痛点或情绪，引发"是我了" |
| 🎯 反常识型 | 红 #ef4444 | 打破常规认知，制造惊讶 |
| 📖 故事型 | 青 #06b6d4 | 用微型叙事吸引注意力 |
| ⚡ 紧迫型 | 黄 #eab308 | 制造时间/机会稀缺感 |

**生成逻辑：**
1. 根据用户主题提取关键词
2. 根据平台调整语气（小红书偏种草、抖音偏快节奏、B站偏深度）
3. 根据内容类型匹配句式风格
4. 每种风格选一个最匹配的模板，填入主题词
5. 为每个结果计算"点击率评分"（80%-98%，基于风格+平台的预定义权重）
6. 为每个结果生成一条"推荐理由"

### 4.3 Hook 卡片
每张卡片展示：
- 左侧色条（按风格颜色）
- 风格标签（圆角药丸）
- 点击率评分（绿色大字）
- Hook 文案（加粗，引号包裹）
- 推荐理由（灰色小字，灯泡emoji开头）
- 操作按钮：📋 复制（点击后变 ✅ 1.5秒恢复）、🤍 收藏（点击变 ❤️）

### 4.4 收藏功能
- 点击卡片上的 🤍 收藏，存入 localStorage
- "收藏夹" Tab 查看所有已收藏Hook
- 支持取消收藏
- 收藏数据格式：`{ id, hook, style, score, reason, platform, contentType, topic, timestamp }`

### 4.5 历史记录
- 每次生成自动保存记录
- "历史记录" Tab 查看最近 20 条生成（按主题分组）
- 点击历史记录可重新加载那一次的10个Hook
- 历史数据格式：`{ id, topic, platform, contentType, hooks: [...], timestamp }`

---

## 5. 交互细节

- **生成时：** 按钮显示 "⏳ 正在生成..." + 卡片区显示骨架屏（3个灰色闪烁卡片占位），模拟 1.2-1.8 秒随机延迟，让用户感觉AI在"思考"
- **复制反馈：** 点击复制 → 图标变 ✅ → 1.5秒后恢复 📋
- **空状态：** 首次打开在结果区显示引导文案 + 3个示例主题快速填入
- **收藏空状态：** "还没有收藏哦，去生成结果页点击 🤍 收藏吧"
- **历史空状态：** "还没有生成记录，输入主题开始吧"
